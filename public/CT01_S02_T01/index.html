T01 Introduction to node class

Fecha de Ultima Revisión 14/05/2018

Descripción

node es una clase fundamental en el framework, este simplemente contiene la información de un punto en el espacio bidimensional, además de el nivel donde se encuentra. El dato de la altura del nodo esta contenida en el nivel donde se encuentra, por lo que podemos acceder a esa información a través de la propiedad level

Variables de entrada

  • r - Posicion bidimiensional de el nodo dentro de un plano.
  • level - Nivel donde se encuentra el nodo.
Parámeteros ¿Obligatorio?   Tipo de Variable Dimensión Opciones Valor por defecto
r Opcional double [1x2] Cualquiera [0 0]
level Opcional double positive integers [1x1] Cualquiera 0

Variables de salida

  • node - Objeto tipo node

Sintaxis

Podemos crear un nodo nulo, simplemente no pasando parámetros

node


ans = 

  node with properties:

        r: [0 0]
    level: 0

también podemos crea un nodo en otra posición

node([1 1])


ans = 

  node with properties:

        r: [1 1]
    level: 0

Por último , podemos especificar el nivel donde queremos crear el nodo

node([4,3],4)


ans = 

  node with properties:

        r: [4 3]
    level: 4

Ejemplos

Podemos crear un nodo en el punto [1 1]

n1 = node([1,1])


n1 = 

  node with properties:

        r: [1 1]
    level: 0

y reasignar su nivel de la siguiente manera,

n1.level = 4


n1 = 

  node with properties:

        r: [1 1]
    level: 4

Por último de la misma forma se puede reasignar la propiedad r

n1.r = [4 1]


n1 = 

  node with properties:

        r: [4 1]
    level: 4

Además podemos crear varios nodos y dibujarlos con ayuda de la función plot

n2 = node([4 4]);
n3 = node([3 5]);
n4 = node([2 1]);
nodes = [n1 n2 n3 n4];
plot(nodes,"*");
xlim([-5 5])
ylim([-5 5])
