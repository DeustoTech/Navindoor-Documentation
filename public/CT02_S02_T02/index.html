T02 velocity parameter

Podemos añadir velocidad a nuestras trayectorias

Creamos una trayectoria

x = -10:0.1:10;
y = -sqrt(100-x.^2) + 0.5*sin(x);
itraj = mat2traj([x' y']);
plot(itraj,'*')

Para agregar velocidad, llamamos a la función velocity

itraj = velocity(itraj)
plot(itraj.t,itraj.v)
xlabel("time(s)");ylabel("velocity(m/s)")
