T03 Introduction to beacon class

Fecha de última revisión 14/05/2018

Descripción

la clase beacon representa los puntos de acceso como pueden ser los router wifi o cualquier tipo de baliza, que nos ayude a posicionar un objeto. Esta clase es una hijo de la clase node, por lo que tiene todas sus funciones y propiedades. La propiedad adicional que tiene esta clase es frecuency, esta propiedad esta pensada para simular señales radioelectricas teniendo en cuenta la frecuencia de propagación. Sin embargo, actualmente no esta implemnetado esta simulación. En la práctica, el objeto beacon Se utiliza para distingurse de otros objetos hijos de node, por las funciones de framework.

Variables de entrada

  • r - Posicion bidimiensional de el nodo dentro de un plano.
  • level - Nivel donde se encuentra el nodo.
  • frecuency - Frecuencia a la que emite este AP-
Parámeteros ¿Obligatorio?   Tipo de Variable Dimensión Opciones Valor por defecto
r Opcional double [1x2] Cualquiera [0 0]
level Opcional double positive integers [1x1] Cualquiera 0
frecuency Opcional double positive integers [1x1] Cualquiera 1

Variables de salida

  • beacon - Objeto tipo beacon

Sintaxis

Podemos crear un beacons nulo, simplemente no pasando parámetros

beacon


ans = 

  beacon with properties:

    frecuency: 1
            r: [0 0]
        level: 0

también podemos crea un nodo en otra posición

beacon([1 1])


ans = 

  beacon with properties:

    frecuency: 1
            r: [1 1]
        level: 0

Podemos especificar el nivel donde queremos crear la AP

beacon([4,3],4)


ans = 

  beacon with properties:

    frecuency: 1
            r: [4 3]
        level: 4

Por último, podemos definir la frecuencia que tiene un nuevo beacon

beacon([4,3],4,2)


ans = 

  beacon with properties:

    frecuency: 2
            r: [4 3]
        level: 4

Otra forma de utilizar las opciones es:

beacon([4,3],'frecuency',4,'level',2)


ans = 

  beacon with properties:

    frecuency: 4
            r: [4 3]
        level: 2

Ejemplos

Podemos crear un beacon en el punto [1 1]

b1 = beacon([1,1])


b1 = 

  beacon with properties:

    frecuency: 1
            r: [1 1]
        level: 0

y reasignar su nivel de la siguiente manera,

b1.level = 4


b1 = 

  beacon with properties:

    frecuency: 1
            r: [1 1]
        level: 4

Por último de la misma forma se puede reasignar las propiedades r y frecuency

b1.r = [4 1]
b1.frecuency = 5


b1 = 

  beacon with properties:

    frecuency: 1
            r: [4 1]
        level: 4


b1 = 

  beacon with properties:

    frecuency: 5
            r: [4 1]
        level: 4

Además podemos crear varios nodos y dibujarlos con ayuda de la función plot

b2 = beacon([4 4]);
b3 = beacon([3 5]);
b4 = beacon([2 1]);
beacons = [b1 b2 b3 b4];
plot(beacons,"*");
xlim([-5 5])
ylim([-5 5])
