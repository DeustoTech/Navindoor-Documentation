T01 Introduction to BeaconFreeSgn

La clase BeaconsFreeSgn, es la evolución de la clase BFSLevel. Mientras que BFSLevel toma como variables de entrada un objeto traj y un objeto level, BeaconsFreeSgn toma como variables de entrada un objeto supertraj y un objeto build. Tiene muchas similitudes, sin embargo el este último, se tiene valores de señal entre plantas

Input Vars  Mandotory/Optional type Options
istraj Mandatory supertraj -
ibuild Mandatory build -
type Mandatory char (‘RSS’,’ToF’,’AoA’)

Ejemplo Básico

clear;
load('data/levels/EI_UD_conn_AP_fakes.mat')
load('data/trajs/straj_in_conn_AP_fakes.mat')
whos

  straj         1x1              142628  supertraj

Antes de nada veamos como son nuestras variables de entrada. La variable building, es un edifício de 5 plantas,

building

Mientras que el objeto straj es una trayectoria dentro del edifício mecionado

straj


straj = 

  supertraj with properties:

Podemos crear una señal de la trayectoria equiespaciada en el tiempo

straj = equitime(straj,0.1);

Generaremos nestra señal de barómetro con la siguiente instrucción

Baro_sgn = BeaconFreeSgn(building,straj,'Baro')

Como en BeaconBasedSgn podemos crear tablas de los tramos

tables = signal2table(Baro_sgn)

Extraemos la tabla del primer tramo recorrido.

tbl1 = tables{1};
tbl1(1:5,:)

Podemos ver gráficamente la señal recibida durante el primer tramo.

plot(tbl1{:,1},tbl1{:,2:end})
xlabel('time(s)');ylabel('Pressure(mmHg)');title('Barometero signal')

Comparandola con el cuarto tramo

tbl4 = tables{4};
hold on
plot(tbl4{:,1},tbl4{:,2:end})
l1 = straj.trajs(1).level; l4 = straj.trajs(4).level;

legend({['level ',num2str(l1)],['level ',num2str(l4)]})

La clase BeaconFreeSgn cuenta con un método que devuelve el valor en cada instante, simpre y cuando el valor del tiempo este definido dentro de la señal

step(Baro_sgn,0)

Obtenemos la señal requerida

step(Baro_sgn,22.3)

Obtenemos la señal requerida en otro instante de tiempo

step(Baro_sgn,22.5)

Hemos encontrado un tramo sin señal, esto ocurre durante los cambios de planta

step(Baro_sgn,32.5)

el número de beacons que recibimos a cambiado, por lo que sabemos que hemos cambiado de planta

% step(Baro_sgn,1000.0)

Por último, si no salimos del rango de definición, generaremos un error.

Señal continua entre plantas

Podemos generar una señal continua de la siguiente manera

% Seleccionamos el t final
tfinal = index2time(straj,"end","end");
% creamos una linea de tiempo
time_line = 0:straj.dt_equitime:tfinal;
% generamos la senal del barometro, con ayuda de arrayfun
pressure_line = arrayfun( @(t)step(Baro_sgn,t),time_line);
% quitamos el anterior grafico y ploteamos el nuevo
hold off
plot(time_line,pressure_line)
xlabel("time(s)")
ylabel("pressure(mmHg)")
grid on
